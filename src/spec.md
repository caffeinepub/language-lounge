# Specification

## Summary
**Goal:** Enable text chat functionality during active voice and video calls in voice rooms.

**Planned changes:**
- Add persistent text chat interface overlay to VoiceRoom component that remains visible during voice calls
- Ensure InRoomChat component stays functional while users are speaking or listening
- Create video call interface with integrated text chat overlay
- Add chat toggle button or permanent chat panel to VoiceRoom page for accessing messages during conversations

**User-visible outcome:** Users can send and receive text messages while participating in voice or video calls, with the chat interface accessible via toggle or overlay without leaving the call.
